My GitHub Repository with Projects Reflecting My Journey into Coding

My GitHub repository showcases a variety of projects that highlight my journey into coding. From web development to configuring 3D printers, each project reflects my learning and growth in different areas of technology. Check out my GitHub at github.com/Sr-0w.

Project Highlights

  • ResumeWebsite: A personal website to showcase my resume and projects.
  • VR_Ping_Pong: A private project focused on creating a VR ping pong game.
  • Ollama-GUI-Python-GTK: A GUI application built with Python and GTK3 for interacting with Ollama.
  • Ender3V2NeoKlipperConfig: Configuration files for the Ender 3 V2 Neo 3D printer using Klipper.
  • OpenCore-X570Gaming-E-Gtx1080: Configuration for OpenCore on a X570 Gaming E motherboard with GTX 1080.
  • YBSpdfFiller: A web-based tool for filling out PDF forms.
