Building My Resume Website

How I built a resume website and its impact on my career

Transitioning from a sales career to IT is an exciting opportunity to leverage my skills in a field I like since my younger age. This resume website project showcases this transition and how it benefits my future career. This article details the project, including technical choices, the learning process, and why it's a great step for my career in IT.

The Project

It began with the goal of creating a personal resume website to showcase my skills, experiences, and projects. The website is hosted on an AWS EC2 instance, uses a custom domain name "snyders.xyz," and is secured with SSL certificates from Cloudflare. Here are the key steps involved:

Developing the Website

I started by designing and developing the website using HTML, CSS, and JavaScript. The website includes sections for my professional experience, education, skills, and projects. The website is fully responsive, adapting to various screen sizes and devices. This approach allowed me to gain a deep understanding of the foundational web technologies without relying on any CMS or frameworks.

The website's back-end is powered by Node.js, which allows me to use the Express framework to manage HTTP requests and responses. PM2, a production process manager for Node.js applications, keeps the application running and restarts it if it crashes. This ensures that the website remains available even in case of unexpected errors or server issues.

This project allowed me to demonstrate my skills in web development, server administration, and cloud computing.

Website Development

The first step was designing and coding the website. Using pure HTML, CSS, and JavaScript, I created a responsive and visually appealing site. This approach ensured that I gained a deep understanding of the foundational web technologies without relying on any CMS or frameworks.

To host the website, I used the AWS Free Tier, which offers a year of free usage for new customers. I launched an EC2 instance, configured the security groups, and set up the necessary software (Node.js and PM2) to run the website.

I purchased the domain name "snyders.xyz" directly through Cloudflare. This setup allowed for easy DNS management and SSL certificate configuration, ensuring my website is secure and loads quickly.

Hosting on AWS EC2

Hosting the website on AWS EC2 involved several steps:

  • Creating an EC2 Instance: Using the AWS Management Console, I launched a new EC2 instance, choosing an Amazon Linux Machine from the free tier.
  • Configuring Security Groups: I set up security groups to allow HTTP and HTTPS traffic to the instance.
  • Setting Up the Server: I installed Node.js and PM2 to manage the web server. PM2 is a production process manager for Node.js applications, which keeps the application running and restarts it if it crashes.

Domain Name Registration and Cloudflare Setup

By purchasing the domain through Cloudflare, I streamlined the process of setting up DNS management and SSL certificates:

  • DNS Management: Cloudflare's platform made it simple to manage DNS records, ensuring my domain pointed correctly to my AWS EC2 instance.
  • SSL Certificate: Cloudflare provides a free SSL certificate, which I configured to ensure all traffic to my site is encrypted.
  • Performance Optimization: Cloudflare's CDN (Content Delivery Network) helps to reduce latency and improve load times for visitors around the globe.

Why This Project Is Important for My Career

This project is a significant step in my transition from sales to IT for several reasons:

  • Practical Experience: Building and hosting a website from scratch provided hands-on experience with web development, server administration, and cloud computing.
  • Problem-Solving Skills: The project involved troubleshooting various issues, from server configurations to domain name settings. This honed my problem-solving abilities, which are essential in IT.
  • Showcasing My Work: The resume website acts as a portfolio, demonstrating my technical skills and ability to manage a complete project lifecycle. It's a powerful tool to showcase what I can do.
  • Learning and Growth: Throughout the project, I learned new technologies and best practices. This continuous learning mindset is crucial for success in the ever-evolving IT field.
  • Professional Online Presence: Having a personalized website with a custom domain name gives me a professional online presence, making it easier for recruiters and hiring managers to find and connect with me.
