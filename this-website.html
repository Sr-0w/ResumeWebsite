THIS WEBSITE

I Built This Portfolio Website to Feature My Projects and Skills Transition

Creating this portfolio website has been a pivotal part of my journey as I transition from a career in sales to network engineering. This website showcases my projects, skills, and the progress I’ve made in acquiring new certifications like CompTIA Network+ and CCNA.

Project Initiation

The project began with designing a layout that would effectively highlight my skills and projects. I chose a modern, responsive design to ensure the site looks great on all devices. The initial setup involved selecting the right tools and technologies to build the site.

Challenges Faced

One of the main challenges was learning web development basics while balancing my studies in network engineering. Integrating dynamic features and ensuring the site is both visually appealing and functional required a lot of research and experimentation.

Learning and Outcomes

This project has significantly improved my understanding of web development and design principles. It has also provided a platform to document and share my progress in network engineering, helping me build a professional online presence.

Conclusion

The 'Portfolio Website' project is a testament to my dedication to learning and growth. It not only highlights my past experiences in sales but also my current journey towards becoming a network engineer. Building this website has been a rewarding experience that bridges my skills from different fields.
